Algebraic simplification

  1. Apply De Morgan: ¬(¬C + B) = C · ¬B.
  2. Substitute: F = (A ⊕ B) · (C · ¬B).
  3. Expand XOR: A ⊕ B = A·¬B + ¬A·B.
  4. Multiply: F = (A·¬B + ¬A·B)·C·¬B = A·¬B·C + ¬A·B·C·¬B.
  5. Since B·¬B = 0, the second term is 0. So
    Final: F = A · ¬B · C
  6. Interpretation: F is true only when A=1, B=0, and C=1 (minterm index 5).
